Para que funcionen las API de google, twitter y facebook el proyecto debe crearse en la carpeta wordpress

  • 1. Crear espacio de trabajo arrow_drop_down

      1. Instalar Xampp en el computador:
        Utilizar la ruta predeterminada: C:\xampp

        https://www.apachefriends.org/es/index.html

      2. Abrir la cónsola de xampp con permisos de administrador

      3. Instalar los Apache y MySql como servicios de windows:
        Hacer Clic en los botones de servicio correspondientes

      4. Habilitar el puerto 443 (SSL):
        Abrir el archivo httpd-ssl.conf de apache C:\xampp\apache\conf\extra

        Y descomentar la linea "Listen 443".

      5. Iniciar los Apache y MySql como servicios de windows:
        Hacer Clic en los botones Start del servicio correspondientes

  • 2. Configurar PHPMyadmin arrow_drop_down

  • 3. Descomprimir el proyecto y configurar conexión a base de datos arrow_drop_down

    1. Descargar el proyecto desed github https://github.com/ljmarquezg/NextU_Recetas

    2. Crear la carpeta wordpress dentro de htdocs y descomprimir dentro de esta el contenido del proyecto.
      C:\xampp\htdocs\wordpress

    3. Verificar que el archivo wp-config.php tenga la información correcta para conectarse al servidor phpmyadmin

      1. DB_NAME: wordpress
      2. DB_User: Usuario de conexión a phpmyadmin
      3. DB_PASSWORD: contraseña de conexión a phpmyadmin
      4. DB_HOST: Nombre del servidor

    4. Iniciar instalaci'on de wordpress


      Abrir el url del proyecto wordpress en el explorador

      https://localhost/wordpress/

    5. Seleccionar el idioma español

    6. Completar la información del sitio y presionar el botón instalar

    7. Al completar la instalación se mostrará un mensaje de éxito

    8. Inicia sesión utilizando los datos ingresados al momento de instalar wordpress

    9. Una vez iniciada la sesión serás redirigido al dashboard

Forzar web a https

Ir a ajustes generale y cambiar el protocolo de conexión por defecto http por https en la opción Dirección del sitio (URL). https://localhost/wordpress/wp-admin/options-general.php

Es necesario forzar el protocolo de conexión a https para evitar inconvenientes y errores a la hora de realizar el inicio de sesión con las redes sociales facebook y google+
